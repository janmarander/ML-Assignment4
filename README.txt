All python files and the data used in this project are uploaded at the public git repository https://github.com/janmarander/ML-Assignment4

See environment.yml for the virtual environment dependencies.

This code was developed in PyCharm. To run the code, simply open main.py in PyCharm using the above-defined environment, and click "run" (you can comment out different sections of the assignment if desired). All plots will be generated along with text files with information for each algorithm. You can also run VIPI.py and QL.py to generate subsets of plots and information.
